Create a Responsive Navbar

Published on March 13, 2025 | By Dev Blog Admin

In today's tutorial, we'll create a responsive navigation bar that adapts to different screen sizes. A well-designed navigation is crucial for user experience, especially as users access websites from various devices.

Why responsive navigation matters

With the increasing number of mobile users, having a navigation that works well on all devices is essential. A responsive navbar collapses into a hamburger menu on smaller screens, saving space while maintaining accessibility.

HTML Structure

We'll start with a simple structure consisting of a logo and navigation links. The key is to include a hamburger menu button that will only be visible on smaller screens.

Styling with CSS

Our CSS will use flexbox for layout and media queries to change the appearance based on screen width. The navigation links will display horizontally on larger screens and vertically in a dropdown menu on smaller ones.

Adding Interactivity with JavaScript

JavaScript will handle toggling the mobile menu when the hamburger icon is clicked. This creates a smooth user experience without requiring page reloads.

script.js 
document.addEventListener('DOMContentLoaded', function() {
    const navbarToggle = document.getElementById('navbar-toggle');
    const navbarLinks = document.getElementById('navbar-links');
    
    navbarToggle.addEventListener('click', function() {
        navbarLinks.classList.toggle('active');
        
        // Animate hamburger to X
        const spans = navbarToggle.querySelectorAll('span');
        spans.forEach(span => span.classList.toggle('active'));
    });
    
    // Close menu when clicking outside
    document.addEventListener('click', function(event) {
        const isNavbar = event.target.closest('.navbar');
        if (!isNavbar && navbarLinks.classList.contains('active')) {
            navbarLinks.classList.remove('active');
            const spans = navbarToggle.querySelectorAll('span');
            spans.forEach(span => span.classList.remove('active'));
        }
    });
});
